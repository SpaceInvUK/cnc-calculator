Nesting — sheet layout

How the parts pack onto sheets. A lot of rules live here (sheet size, margins, grain, rotation, offcuts) — this is where they are visible and editable, instead of hidden.
Sheet rule: a sheet only holds parts with the same Material + Thickness + Sheet size. Anything different starts a new sheet. You can change Material and Size right on each sheet header below (like changing the size in the old app — now material/thickness too).
Manual control: click a part on the sheet to select it, then press 9 / 0 to rotate 90° — same as today. (Drag-to-move planned.)
